# Prerequisites

The Python Plugin is available at the OpenTAP website (opens new window). This plugin requires the following:

  • OpenTAP version 9.15.4 or later

    • Open source development tools can be downloaded from OpenTap.io (opens new window)
    • Commercial evelopment tools can be download from Keysight (opens new window).

  • Python Python 3.7 or later. Download from python.org (opens new window).

  • Supported Operative Systems: Windows 7 or newer, Ubuntu 20.04 or equvilalent or newer, MacOS 10

Note that the platform versions of OpenTAP and Python must match: 64-bit OpenTAP with 64-bit Python; 32-bit Keysight Test Automation with 32-bit Python. For MacOS, the python architecture must match the dotnet architecture, so if dotnet is Apple Silicon (M1) then must python be as well.
